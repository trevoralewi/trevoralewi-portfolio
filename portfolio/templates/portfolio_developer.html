About the Project

My Personal Portfolio Website is an interactive Django web application showcasing my software development projects, technical skills, and professional experiences. Designed with a responsive and mobile-friendly approach, the website uses Bootstrap to ensure an engaging and consistent user experience across different devices. Hiring managers and potential clients can easily explore my work and connect with me through the dedicated contact section, which includes links to my LinkedIn, GitHub, and email.

Key Features

  • Responsive and mobile-friendly design using Bootstrap for cross-device compatibility
  • Engaging project showcase with in-depth descriptions and imagery
  • Contact section with email and direct links to LinkedIn and GitHub profiles for convenient communication
  • Cloud-based hosting on Heroku, a versatile and scalable Platform as a Service (PaaS) solution
