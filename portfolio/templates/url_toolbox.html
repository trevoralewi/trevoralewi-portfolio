Developer Details

URL Shortener

The URL Shortener tool is a convenient and efficient way to condense long website URLs into shorter formats. Built using Python, this tool leverages the Pyshorteners library to handle URL shortening functionality. The user interface is developed with the help of a custom Tkinter library, called customtkinter, which simplifies the process of creating the necessary input fields and buttons in the GUI. Upon pasting a URL into the designated entry box and clicking the 'Shorten URL' button, the application generates a shortened version of the provided URL.

QR Code Generator

The QR Code Generator tool empowers users to convert any URL into a scannable QR code with ease. Developed in Python, this tool employs the QRCode library to create QR codes from URLs. The customtkinter library is utilized to construct the graphical user interface, which enables users to paste the desired URL into the appropriate entry box and generate a QR code by clicking the 'Create QR Code' button. To ensure proper functionality, users must provide a valid URL as input.

Digital Flyer Generator

The Digital Flyer Generator is a versatile marketing tool designed for businesses, individuals, and special events. This Python-based tool utilizes the Pillow (PIL) library for image manipulation and the NumPy library for numerical operations. The customtkinter library is used to create an interactive GUI, which allows users to upload photos, select templates, and input relevant information, such as title, description, and contact details. Text wrapping is handled by the textwrap module, ensuring that the content fits within the designated flyer area. Users can preview different templates, compare color schemes, and generate the final flyer by clicking the 'Generate' button. The generated flyer can then be saved and shared as needed.
