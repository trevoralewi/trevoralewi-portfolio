About the Project

MyMovie Database is a comprehensive web application designed for movie enthusiasts to browse, search, and discover a vast selection of movies. Offering a seamless and intuitive user experience, this platform displays information such as movie ratings, plot summaries, cast details, and more. By leveraging the TMDB API and Heroku Scheduler, MyMovie Database automates the process of updating its MySQL database with fresh movie data, ensuring that users always have access to the most current information.
